###Задача:
```
Напишите класс на языке PHP, реализующий четыре арифметических действия над комплексными числами.
Напишите модульные тесты к этому классу.
```
---
###Installation:
```
$ git checkout git@github.com:Kpeamop/test_uniteller.git
$ cd test_uniteller
$ composer install 
```
---
